# yuki-portfolio

macOS Finder-style personal portfolio — Next.js 14, Framer Motion, Zustand.

## Quick start

```bash
npm install
npm run dev
# open http://localhost:3000
```

## Customize your content

Edit `src/data/portfolio.ts` — all your personal info, projects, and tech stack live there.

## Deploy

```bash
npm run build
# Deploy the .next folder to Vercel, or run `npx vercel`
```

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| ⌘1 / ⌘2 / ⌘3 | Icon / List / Column view |
| Space | Quick Look selected project |
| Esc | Close Quick Look |
| ↑ ↓ | Navigate sidebar |
| ⌘K | Focus search |
